## Goal

Provide a 1-2 paragraph summary of the current conversation and work state.

## Instructions

When this skill is invoked:

1. Review the most recent user request, completed work, pending work, and any blockers.
2. Produce a maximum of two paragraphs.
3. Keep it concise and specific.
4. Mention specifically what is done and what is next when applicable.
5. Do not ask follow-up questions unless the user explicitly asks for questions.

## Output Rules

- Output only the paragraphs.
- No bullets, headers, or extra commentary.
